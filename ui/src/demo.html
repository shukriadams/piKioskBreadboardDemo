Attached to this display is a Raspberry Pi, and attached to that is a small controller board. Click Forward on the controller to proceed.

How?

For this display, a controller that could advance or reverse slides was required. One was put together from standard parts, and a program written to handle button clicks. In this case, clicking blinks a light, and calls an Express server via REST.

Express is a NodeJS web server, also running on this Pi. Express serves the demo page you're reading now. Yes, this is a plain old webpage. Express then uses a websocket to pass along button clicks to the page.

When the button signal reaches the webpage, it triggers a jQuery event which scrolls the display back or forward.

Click Forward for more (hint : you can also click Back)

Why?

Anything that can send trigger a signal to a Pi can be converted into a browser event. Movement in front of a camera, standing on a pressure plate, touching electrosensitive paper, it's all good.

Similarly, the electronic content on display here is just a regular website - css, javascript and images. No internet connection is required (everything runs locally on the Pi).
